# Superstore Sales Analysis using Exploratory Data Analysis (EDA)

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Superstore Sales dataset to understand sales performance, profit trends, customer behavior, and business insights. The notebook includes data cleaning, preprocessing, visualization, and statistical analysis to prepare the dataset for further machine learning or business intelligence tasks.

---

## Objectives

- Import and explore the Superstore dataset
- Perform initial data inspection
- Handle missing values and inconsistencies
- Remove duplicate records
- Detect and treat outliers
- Clean categorical data
- Perform exploratory data analysis
- Generate meaningful business insights using visualizations

---

## Steps Performed

### 1. Import Required Libraries
Loaded all necessary Python libraries for data analysis and visualization.

### 2. Load Dataset
Imported the Superstore dataset into a Pandas DataFrame.

### 3. Data Inspection
- Dataset shape
- Data types
- Summary statistics
- Null value analysis

### 4. Data Cleaning
- Handling missing values
- Checking inconsistencies
- Data formatting

### 5. Duplicate Removal
Identified and removed duplicate records to improve data quality.

### 6. Outlier Detection
Detected outliers using visualization techniques such as Boxplots and applied appropriate treatment methods where necessary.

### 7. Data Consistency
Standardized categorical values and ensured consistency across the dataset.

### 8. Exploratory Data Analysis (EDA)
Performed visual and statistical analysis including:
- Sales Distribution
- Profit Distribution
- Category-wise Analysis
- Sub-category Analysis
- Regional Performance
- Customer Segment Analysis
- Shipping Mode Analysis
- Correlation Analysis

---

##  Key Insights

- Sales distribution is positively skewed with several high-value transactions.
- Outliers significantly affect sales and profit distributions.
- Some product categories contribute more to revenue but not necessarily to profit.
- Regional sales performance varies considerably.
- Customer segments exhibit different purchasing behaviors.
- Certain sub-categories generate losses despite high sales.
